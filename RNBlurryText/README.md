# react-native-blurry-text (WIP)

react native native bridge to show blur text

## Installation

```sh
npm install react-native-blurry-text
```

## Usage

```js
import BlurryText from "react-native-blurry-text";

// ...

const result = await BlurryText.multiply(3, 7);
```
